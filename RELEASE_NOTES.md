# Release Notes

## 0.2.0 - 2026-03-31

원 논문형 사진 프라이밍 구조에 맞추어 실험 프로토타입을 재구성했습니다.

### Added

- 조건별 로컬 자극 이미지 5장씩 사용하도록 구성
- 사진 자극 5초 제시와 1~7점 귀여움 평정 흐름 추가
- 시나리오 전 전환 안내 화면 추가
- 도덕적 시나리오 3개 텍스트 평정 단계 추가
- 자극별 평정과 시나리오별 평정을 포함한 결과 JSON 정리
- 실험 종료 시 CSV 자동 다운로드 추가

### Changed

- 기존 `영상 프라이밍 + 3D 시뮬레이션` 구조를 제거
- 실험 흐름을 `사진 평정 -> 텍스트 시나리오 평정` 구조로 변경
- 실험용 전체 레이아웃과 상태 패널을 사진 프라이밍 중심으로 재설계
- 불필요한 영상 자산과 3D 의존성을 제거
- 조건 수동 선택을 제거하고 실험 시작 시 무작위 배정으로 변경

### Current Experiment Shape

- 보통 자극 5장 또는 귀여운 자극 5장 중 한 조건만 제시
- 각 자극은 5초 동안 표시
- 자극 귀여움 평정은 1~7점
- 시나리오 도덕성 평정은 1~7점

### Known Limitations

- 자극 이미지는 프로토타입용 세트이며 원 논문의 검증 자극과 동일하지 않음
- 데이터 저장은 브라우저 CSV 다운로드만 지원
- 참가자 관리, 랜덤 배정 저장, 외부 설문 시스템 연동은 아직 없음
