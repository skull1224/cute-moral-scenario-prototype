# Cute Prime Moral Judgment Prototype

Zhang & Ye (2023)의 실험 구조를 기준으로 만든 사진 프라이밍 기반 온라인 실험 프로토타입입니다.

현재 버전은 참가자가 실험 설명을 읽고 시작하면 내부적으로 조건이 무작위 배정되고, 같은 조건의 사진 5장을 각 5초씩 보며 귀여움을 1~7점으로 평정한 다음, 도덕적 시나리오 3개를 읽고 도덕성을 1~7점으로 평정하도록 구성되어 있습니다.

## 현재 실험 흐름

1. 실험 설명
2. 사진 프라이밍 5장
3. 전환 안내
3. 텍스트 시나리오 3개 평정
4. 결과 로그 확인

## 시작 방식

- 소개 화면의 `실험 시작` 버튼으로 시작합니다.
- 조건은 참가자에게 보이지 않게 내부적으로 무작위 배정됩니다.

## 사진 프라이밍 단계

- 조건별 자극 수: 5장
- 제시 시간: 장당 5초
- 평정 척도: 1~7점
- 질문: `이 사진이 얼마나 귀여웠나요?`

응답 방법:

- 키보드 `1`~`7`
- 화면의 점수 버튼 클릭

현재 자극 이미지는 프로토타입용 로컬 세트입니다.

- 귀여운 자극: `public/stimuli/cute/`
- 보통 자극: `public/stimuli/neutral/`

## 시나리오 평정 단계

사진 평정이 끝나면, 앞선 과제와는 별개의 판단 과제로 넘어간다는 전환 안내 화면이 먼저 제시됩니다.

논문 부록을 바탕으로 다음 3개 시나리오를 텍스트로 제시합니다.

- 시험 부정행위
- 대필 논문 제출
- 이력서 허위 기재

질문:

- `주인공의 행동은 얼마나 도덕적이라고 생각하나요?`

평정 척도:

- 1 = 매우 비도덕적
- 7 = 매우 도덕적

## 실행

```bash
npm install
npm run dev
```

프로덕션 빌드:

```bash
npm run build
```

미리보기:

```bash
npm run preview
```

## 주요 파일

- 앱 흐름과 평정 로직: `src/main.js`
- 레이아웃과 스타일: `src/style.css`
- 귀여운 자극 이미지: `public/stimuli/cute/`
- 보통 자극 이미지: `public/stimuli/neutral/`

## 현재 한계

- 자극 이미지는 검증된 원본 자극이 아니라 프로토타입용 로컬 세트입니다.
- 결과 화면에 도달하면 CSV 파일이 브라우저에서 자동 다운로드됩니다.
- 서버 저장은 아직 없고 브라우저 다운로드만 제공합니다.
- 참가자 관리나 외부 설문 시스템 연동은 아직 없습니다.
